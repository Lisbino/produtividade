# Controle de Produtividade - Analista Judiciário

Aplicativo web para controle pessoal de produtividade na elaboração de minutas judiciais.

## Descrição

Este aplicativo permite o registro e acompanhamento da produção de minutas de sentenças, despachos e decisões judiciais, oferecendo estatísticas diárias e histórico completo.

## Funcionalidades

- Registro de minutas por tipo (sentença, despacho, decisão)
- Classificação por complexidade
- Cronômetro integrado para medição de tempo
- Estatísticas diárias de produtividade
- Histórico completo com possibilidade de exclusão
- Armazenamento local dos dados no navegador

## Como Utilizar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Preencha o formulário com os dados da minuta elaborada
3. Utilize o cronômetro para medir o tempo de elaboração (opcional)
4. Clique em "Registrar" para salvar os dados
5. Visualize as estatísticas e o histórico na mesma página

## Armazenamento de Dados

- Os dados são armazenados localmente no navegador (localStorage)
- Os registros permanecem disponíveis mesmo após fechar o navegador
- Para acessar os mesmos dados em outro dispositivo, é necessário exportar e importar os dados

## Limitações

- Os dados são armazenados apenas no navegador onde foram registrados
- Não há sincronização automática entre dispositivos
- Recomenda-se fazer backup periódico dos dados

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage para persistência de dados
